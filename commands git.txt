GIT COMMANDS

1) git status
2) git add index.html  \ git add .   (если добавить все)
3) git status
4) git rm --cashed index.html (удаляем)
5) создаем файл .gitignore для того, что бы добавить тот или инной файл не включать в git	
6) git branch test (для создания веток)(git checkout -b new) \ чтобь удалить git branch -D test \ чтобы просмотреть git branch
7) если перекочится на ветку то git checkout test
8) git merge test (объеденить две ветки)
9) git push чтобы закинутьь на git
10) перед началом работ с git если есть второй пользователь и он делал изменения то 
лучше скачть их командой git pull
